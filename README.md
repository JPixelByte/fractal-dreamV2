## 🔬 The Computer Science & Mathematical Core

### 1. Number Theory (Counting Prime Sheep)
The engine generates its spatial positioning by hunting down Prime Numbers using an optimized trial division routine. The distribution properties of primes prevent the stars and characters from traveling in rigid patterns, structuring them into loose, organic star clusters instead.

### 2. Neuro-Acoustic Entrainment (The Multi-Channel Crossover Breakthrough)
By expanding the dual-oscillator architecture to completely bypass buggy browser panning engine hardware routing panics, the synthesizer now incorporates a highly sophisticated hybrid routing configuration:
*   **Original Single-Tone Modes (100Hz base):** Standard pure sine wave gap separation (2Hz Delta / 6Hz Theta) optimized for old computer soundcards, laptops, and headphone lines.
*   **Subwoofer Active Layered Modes (250Hz + 100Hz base):** A proprietary multi-layered stack designed to bypass hardware active speaker crossovers. By layering a high-clarity 250Hz carrier wave over a deep 100Hz sub-bass tone, the engine successfully prevents hardware phase-cancellation muffling.
*   **The 5.0x Native Voltage Amplifier Matrix:** Applies a raw software boost curve directly onto the independent gain nodes to guarantee full acoustic masking relief through advanced high-end subwoofer components.

### 3. DSP Precipitation Synthesis (Procedural Rain)
To create realistic weather elements without an `.mp3` footprint, a custom random transient injector buffer scans a live 44.1kHz audio data array. It injects mathematically decaying exponential curves ($Math.exp$) directly onto the white/pink power spectral distributions, translating raw floating-point calculations into distinct, comforting sound impulses that mimic rain striking glass.
